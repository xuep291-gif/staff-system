## 学生端快速上手

| 步骤 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 1. 登录 | POST | `/api/student/auth/login-code` | 手机号+学号+验证码 |
| 2. 查账单 | GET | `/api/student/bill/list` | 需在 Header 带 Token |
| 3. 选宿舍 | GET | `/api/student/dorm/buildings` | 查楼栋列表 |

### 登录示例

```javascript
// 1. 先发验证码
fetch('https://xxxx-xxx.ngrok-free.dev/api/student/auth/send-code', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ phone: '手机号', sid: '学号' })
})

// 2. 然后登录（验证码开发环境固定 8888）
const res = await fetch('https://xxxx-xxx.ngrok-free.dev/api/student/auth/login-code', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ sid: '学号', phone: '手机号', code: '8888' })
})
const data = await res.json()
// data.data.token → JWT token
// data.data.student → 学生信息

// 3. 后续请求带上 token
fetch('https://xxxx-xxx.ngrok-free.dev/api/student/bill/list', {
  headers: { 'Authorization': 'Bearer ' + token }
})
```

## 教师端快速上手

| 步骤 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 1. 登录 | POST | `/api/v1/auth/login/password` | 工号+密码 |
| 2. 首页 | GET | `/api/v1/dashboard/teacher` | 仪表盘数据 |
| 3. 学生列表 | GET | `/api/v1/students/search` | 搜索学生 |

```javascript
// 教师登录
const res = await fetch('https://xxxx-xxx.ngrok-free.dev/api/v1/auth/login/password', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ account: '1001', password: '123456' })
})
const data = await res.json()
// data.data.accessToken → JWT token
```

## 通用规则

- 所有请求头：`Content-Type: application/json`
- 需要登录的接口头：`Authorization: Bearer <token>`
- 成功返回：`{ "code": 0, "message": "success", "data": {...} }`
- 失败返回：`{ "code": 非0, "message": "错误描述" }`
