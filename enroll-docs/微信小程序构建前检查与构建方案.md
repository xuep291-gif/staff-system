## 四、样式兼容性检查

### 🔴 严重：`vh`/`vw` 单位（45+ 处）

微信小程序**不支持** `vh`/`vw` 单位，必须替换。

**涉及文件**（部分列表）：

| 模式 | 文件数 | 典型位置 |
|------|--------|---------|
| `min-height: 100vh` | ~35 个 | 几乎所有页面的 `.page` 类 |
| `max-width: 100vw` | 4 处 | `App.vue` |
| `calc(100vh - 104rpx)` | 1 处 | `finance/urge/index.vue` |
| `max-height: 70vh` / `50vh` | 2 处 | `SBottomSheet.vue` |
| `height: 100vh` | 2 处 | `globalStyle.scss` |

**修复方案**：`100vh` → `100%`（依赖父级高度）或通过 JS 计算。小程序中 `page` 标签默认高度 100%，所以 `height: 100%` + `flex: 1` 的组合可以替代大部分 `100vh`。

### 🟡 中等：`position: fixed`（23+ 处）

微信小程序中 `position: fixed` 在祖先有 `transform`/`filter`/`will-change` 时失效。

**涉及组件/页面**：

| 位置 | 用途 |
|------|------|
| `SNavBar.vue` | 导航栏固定 |
| `STabBar.vue` | 底部导航固定 |
| `SBottomSheet.vue` | 底部弹出面板遮罩 |
| ~18 个页面的 `.ovl` 类 | 弹窗遮罩层 |

**修复方案**：确保 fixed 元素的祖先没有 transform；对于弹窗遮罩，可考虑使用 `<cover-view>` 或小程序原生组件。

### 🟡 中等：`px` 与 `rpx` 混用

| 文件 | 问题 |
|------|------|
| `static/rssGlobalStyle/rssGlobalStyle.scss` | 大量 `px` 与 `rpx` 混用 |
| `components/dynamic-list/listItem/messageItem.vue` | `px` 和 `rpx` 在同一组件中混用 |
| `styles/globalStyle/globalStyle.scss` | SCSS 变量使用 `px` |
| `uni.scss` | uni-app 变量声明使用 `px` |

**修复方案**：`border` 保留 `1px` 是正常的，但 `padding`/`margin`/`font-size` 应统一为 `rpx`。

### 🟢 低风险

| 项目 | 状态 |
|------|------|
| `z-index: 9999` | 1 处（login.vue），需关注原生组件层级 |
| CSS Grid | 1 处（messageItem.vue），旧版基础库支持有限 |
| `env(safe-area-inset-bottom)` | 1 处（SBottomSheet.vue），需回退方案 |
| `::before`/`::after` | 未使用 ✅ |
| `backdrop-filter`/`filter` | 未使用 ✅ |
| 嵌套 scroll-view | 未发现 ✅ |
| scoped 中的 @import | 未使用 ✅ |

### CSS 动画

`@keyframes`（`pulse-dot`、`line-glow`）在 6 个文件中重复定义。小程序支持 CSS 动画，但建议提取到公共文件减少体积。

## 六、导航与返回按钮检查

### 6.1 `uni.navigateBack` 缺少栈检查（~20 处）

以下页面的 `navigateBack()` 没有先检查 `getCurrentPages().length > 1`：

| 模块 | 页面 |
|------|------|
| 教师端 | `doc-review`、`doc-done`、`aid-review`、`loan-review`、`non-dorm-review`、`room-change-review` |
| 财务端 | `collect-detail`、`receipt-detail`、`onsite`、`aid-review`、`loan-review`、`refund-review` |
| 政务端 | `aid-review`、`aid-final-review`、`loan-review`、`loan-final-review`、`non-dorm-review`、`dorm-review` |

**风险**：通过深度链接打开页面时，页面栈为空，`navigateBack()` 静默失败，用户无法返回。

**修复方案**：
```javascript
const pages = getCurrentPages()
if (pages.length > 1) {
  uni.navigateBack()
} else {
  uni.reLaunch({ url: '/pages/xxx/home/index' })
}
```

### 6.2 `uni.navigateBack({ delta: 10 })` 

`utils/tools.js` 第 262 行使用 delta: 10 返回，页面栈深度几乎不可能达到 10，会静默失败。
