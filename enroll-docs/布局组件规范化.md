# 布局组件规范化

> 基于 student-miniapp-v3.1(1) 原型 CSS 结构 · 逐组件审计 · 输出小程序兼容规范
> 版本 v3.0 · 2026-05-22

---

## 〇、小程序兼容性总览

| 属性 | 小程序 | 本规范处理 |
|------|--------|-----------|
| `gap`（flex/grid） | 不支持 | → `> * + * { margin }` 或 border 分割 |
| `:has()` | 不支持 | → Vue `:class` 控制 |
| `inset` | 部分不支持 | → 展开为 top/right/bottom/left |
| `aspect-ratio` | 不支持 | → `padding-bottom: 100%` |
| `accent-color` | 不支持 | → 小程序原生组件 |
| `px` 单位 | H5 可用，小程序建议 rpx | → 全部换算为 rpx |
| `transition` | 基础属性可用 | → 保留 |
| `pointer-events` | 不支持 | → `visibility: hidden/visible` 替代 |
| `calc()` | 部分支持（简单运算可用） | → 复杂 calc 改固定值 |
| `overflow-y: auto` | 需用 `<scroll-view>` 组件 | → 标记改用 scroll-view |
| `display:flex` | 支持 | → 保留 |
| `display:grid` | 支持（不含 gap） | → gap 用 border 替代 |

| 章节 | 组件 | 兼容性问题 | 处理 |
|------|------|-----------|------|
| 1 | Design Tokens | 无 | 直接复用 |
| 2 | Shell/Phone/StatusBar | 预览框架 | uni-app 项目不需要 |
| 3 | Screens | `inset` `gap` `pointer-events` `overflow-y` | 展开 inset，gap 改 margin，pointer-events 改 visibility，overflow-y 改 scroll-view |
| 4 | NavBar | `gap` | `> * + *` 替代 |
| 5 | Banner | 无 | px → rpx |
| 6 | FloatCard | `letter-spacing` | 保留（降级忽略） |
| 7 | Cards | `text-transform` | 删除中文字段 |
| 8 | Badges | `gap` | `> * + *` 替代 |
| 9 | Tabs | `::-webkit-scrollbar` | 小程序自动忽略 |
| 10 | ListItems | `gap` ×2 | `> * + *` 替代 |
| 11 | StepIndicators | 无 | px → rpx |
| 12 | Checklist | `gap` | `> * + *` 替代 |
| 13 | DormGrid | `gap` `:hover` | margin 替代，hover 保留 |
| 14 | BuildingFilter | `gap` `::-webkit-scrollbar` | `+` 相邻选择器，用 scroll-view |
| 15 | RoomCard | `gap` | `> * + *` 替代 |
| 16 | UploadArea | `gap` `aspect-ratio` `calc` | margin + 固定高度替代 calc |
| 17 | Form | `gap` ×3 `resize` | `> * + *` 替代，删除 resize |
| 18 | Buttons | `gap` | `> * + *` 替代 |
| 19 | AlertBars | `gap` | `> * + *` 替代 |
| 20 | InfoRows | 无 | px → rpx |
| 21 | PaymentMethods | `gap` | `> * + *` 替代 |
| 22 | Result | `gap` | `> * + *` 替代 |
| 23 | BillItems | 无 | px → rpx |
| 24 | Countdown | `gap` `font-variant-numeric` | `+` 相邻选择器 |
| 25 | Tabbar | `gap` `calc` | gap 改 `> * + *`，calc 改固定值 |
| 26 | BottomSheet | `inset` `gap` `pointer-events` `overflow-y` | 展开 inset，gap 改 margin，pointer-events 改 visibility |
| 27 | BindScreen | `gap` ×2 `overflow-y` | gap 改 `> * + *`，overflow-y 改 scroll-view |
| 28 | Misc | 无 | px → rpx |

> **核心替换模式**：`gap` → `> * + * { margin }` · `inset` → 显式四向 · `pointer-events` → `visibility` · `aspect-ratio` → 固定高度 · `calc()` → 固定值 · `overflow-y` → `<scroll-view>` · `px` → `rpx`（×2）
