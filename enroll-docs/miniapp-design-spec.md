## 02 字体规范 Typography

**字体栈：** `-apple-system, BlinkMacSystemFont, 'PingFang SC', 'Helvetica Neue', sans-serif`

| 层级 | 字号 | 字重 | 颜色 Token | 用途示例 |
|------|------|------|-----------|---------|
| 大数字 | 22px | 700 | `#2B6CB0` / `#D97706` | 待缴金额、完成人数 |
| 页面标题 | 18px | 700 | `N-900` | 用户姓名 |
| 卡片标题 | 15px | 600 | `N-900` | 报到步骤、功能卡标题 |
| 功能入口 | 13px | 600 | `N-700` | 待办事项标题、入口文字 |
| 正文 | 13px | 400 | `N-700` | 描述文案、通知内容 |
| 辅助说明 | 11px | 400 | `N-500` | 时间戳、补充说明 |
| Section Label | 10px | 600 | `N-400` | 分区标签（uppercase + letter-spacing 0.5px）|

> **规则：** 字重仅使用 400 / 600 / 700 三档。禁止使用 300（过细）和 900（过重）。

## 03 间距规范 Spacing

**基础网格：4px**  
所有间距值必须为 4 的倍数。

| Token | 值 | 用途 |
|-------|-----|------|
| `xs` | 4px | 图标与文字间距、Badge 内边距 |
| `sm` | 8px | 列表项行内间距 |
| `md` | 12px | 卡片内元素间距 |
| `lg` | 16px | 卡片内边距（padding） |
| `xl` | 20px | 卡片间距（margin-bottom） |
| `2xl` | 24px | 页面区块间距 |

**禁用值：** 5px、6px、7px、9px、10px、11px、13px、14px — 破坏 4px 节奏感。

## 05 图标规范 Icons

**图标库：** Tabler Icons（outline 风格，禁止使用 filled 变体）

| 尺寸 | 适用场景 |
|------|---------|
| 22–24px | TabBar 导航图标、Banner 操作图标 |
| 18–20px | 功能入口宫格图标（44×44 背景托内）|
| 14–16px | 行内辅助图标（列表、表单）|

**图标色彩规则：**
- 导航选中态：`#2B6CB0`（品牌蓝）
- 功能图标：跟随语义色（缴费=红、核验=绿、宿舍=蓝）
- 禁用/非选中：`#9CA3AF`

## 08 卡片规范 Cards

### 层级体系（由高到低）

**① Banner 卡片**
- 背景：`#2B6CB0`（品牌蓝实色，禁止渐变）
- 位置：页面顶部，高度约 88–100px
- 用途：身份信息、批次名称展示

**② 浮岛卡片（首屏数据卡）**
- 背景：`#FFFFFF`
- 圆角：`14px`
- 阴影：`box-shadow: 0 4px 16px rgba(0,0,0,.10), 0 1px 4px rgba(0,0,0,.06)`
- 位置：`margin-top: -28px` 叠压 Banner，产生悬浮层次
- 用途：进度概览、3 列关键数据

**③ 内容卡片（通用）**
- 背景：`#FFFFFF`
- 圆角：`14px`
- 边框：`1px solid rgba(0,0,0,.07)`（默认）/ `1px solid #FCA5A5`（紧急状态）
- 阴影：`box-shadow: 0 1px 3px rgba(0,0,0,.06)`

**④ 内嵌格（功能宫格）**
- 背景：`#F8F9FB`
- 圆角：`12px`
- 边框：`1px solid #F0F2F5`
- 无阴影
- 用途：快捷入口 3 列宫格

## 10 设计规则 Design Rules

1. **4px 基础网格** — 所有间距为 4 的倍数，禁用 5/6/7/9/10px
2. **触控最小尺寸** — 可点击元素高度 ≥ 44px，图标按钮 ≥ 32×32px
3. **色彩即语义** — 蓝=可交互/主操作，绿=成功，橙=警告，红=危险，紫=绿通
4. **禁止渐变** — Banner 与按钮均为实色，禁止 `linear-gradient`
5. **卡片层级** — Banner › 浮岛（高阴影）› 内容卡片（低阴影）› 内嵌格（无阴影）
6. **图标统一** — Tabler outline 风格，同一页面图标视觉重量保持一致
7. **对比度** — 正文 `#374151` on 白底 ≥ 7:1，辅助文字 ≥ 4.5:1（WCAG AA）
8. **Banner 文字** — 白色 `#FFF` on `#2B6CB0` 对比度 4.8:1（满足 AA）；副标题用 `rgba(255,255,255,.65)` 区分层级
9. **字重三档** — 仅用 400（正文）/ 600（标题、按钮）/ 700（大数字、姓名）
10. **3 列上限** — 宫格布局最多 3 列（375px 屏幕下每格约 103px，保证 44px 触控目标）
